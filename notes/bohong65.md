第1章 导论

| 核心概念 | 要点 |
| --- | --- |
| 宏观经济学研究对象 | 经济增长（收入、就业）、经济波动（失业、通胀）、经济政策（政府干预） |
| 三类宏观经济模型 | ①经济增长模型 ②长期总供给-总需求模型 ③短期总供给-总需求模型 |
| 总供给曲线(AS) | 价格与产量的关系；长期垂直、短期平坦/向上倾斜 |
| 总需求曲线(AD) | 产品市场与货币市场同时均衡时价格与收入的关系 |
| 时间维度划分 | 特长期(very long run)、长期(long run)、中期(medium run)、短期(short run) |
| 菲利普斯曲线 | 失业率与通胀率的交替关系 |
| 产出缺口 | 实际产出与潜在产出的差额 |
| 潜在产出 | 充分就业时的产出水平 |

第2章 国民收入核算

| 核心概念 | 定义/公式 |
| --- | --- |
| GDP（国内生产总值） | 地域概念；最终产品与服务的市场价值 |
| GNP（国民生产总值） | 国民概念；本国国民拥有的生产要素生产的价值 |
| NDP（国内生产净值） | GDP - 折旧 |
| NI（国民收入） | NNP - 企业间接税 = 工资+利息+租金+利润 |
| PI（个人收入） | NI - 企业利润 + 转移支付 + 利息收入 |
| DI（个人可支配收入） | PI - 个人所得税 |
| 支出法恒等式 | Y = C + I + G + NX |
| 收入法 | GDP = 工资+利息+租金+利润+间接税+转移支付+折旧 |
| 名义GDP vs 实际GDP | 当期价格 vs 基期价格 |
| GDP紧缩指数 | 名义GDP/实际GDP |
| CPI（消费价格指数） | 固定商品篮子的价格变化 |
| PPI（生产价格指数） | 生产环节的价格，通常领先CPI |
| 通货膨胀率 | π = (Pt - Pt-1)/Pt-1 |
| 实际利率 | 名义利率 - 通货膨胀率 |
